 # Herramientas MCP disponibles

 Estas herramientas se exponen al cliente MCP y son ejecutadas vía `CallTool`.

 ## Máquinas virtuales (QEMU)

 - `listVMs(node)`
 - `getVM(node, vmid)`
 - `startVM(node, vmid)`
 - `stopVM(node, vmid)`
 - `restartVM(node, vmid)`
 - `deleteVM(node, vmid)`

 ## Contenedores (LXC)

 - `listContainers(node)`
 - `getContainer(node, vmid)`
 - `startContainer(node, vmid)`
 - `stopContainer(node, vmid)`
 - `restartContainer(node, vmid)`
 - `deleteContainer(node, vmid)`
