 # Inicio rápido

 ## Instalación

 ```bash
 git clone <repository-url>
 cd proxmox-mcp
 npm install
 npm run build
 ```

 ## Desarrollo

 ```bash
 npm run dev
 ```

 ## Producción

 ```bash
 npm run build
 npm start
 ```

 Continúa con [Configuración](./configuration.md) para definir variables de entorno y autenticación.
